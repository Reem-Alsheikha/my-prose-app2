# My Prose App

A simple Java Maven project that demonstrates the use of the ProseBuilder library.

## Features

- Generates simple prose from registered sentence implementations.
- Uses Maven as build system.
- Includes GitHub Actions for automated testing.
- Published as a GitHub Package.

## Requirements

- Java 21
- Maven 3.9+

## Build

```bash
mvn clean install
```

## Run

```bash
mvn exec:java
```
